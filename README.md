# Random Password Genearator:
A password generator that can randomly generate passwords based on the given criteria by the user which is built in Python.

## Features:
**Error Handling:** It utilizes try-except block to prevent crashes from invalid data types of password length (e.g non-integer data for password length.)
**Criteria Checks:** It checks the input conditions at once and using while loop which prompts re-entry without terminating the execution of the program.
**Stores in Text File:** The randomly generated passwords can also be saved on text files if the user desires to do so.

## Technologies Used:
Python 3
Built-in Libraries (string, random)

''' bash
python pass_gen.py
